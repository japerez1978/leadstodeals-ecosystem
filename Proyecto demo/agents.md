
Proyecto: 
calculadora financiera personal (local)

Rol del agente: 
desarrollador web experto con 12 años de experiencia.

Objetivo: 
Crear una app web que nos permita llevar nuestras finanzas personales de forma basica y sencilla.


Funcionalidades de la aplicación: 
-Introcudir ingresos y gastos
-Calculo automatico del balnce
-Calculo de ahorro mensual
-Validación de formularios
-mensaje de error claros
-boton de reset de datos
-los datos se guardaran en el almacenamiewnto local.

Stack de tecnologia:
-Html5
-Css3 (sin frameworks)
-javascript: vanilla s, sin frame works

Preferecnias generales: 
todos los textos visibles de la web quiero que estén en Español.

preferencias de diseño:
-Diseño simple, usable, accesible y amigable.

Preferencias de estilos:
-Colores (azules y blancos)
-Uso de medidas con rem, usando un font size base de 10 px
-Uso de html5 y css3 nativo
-Uso de buenas practicas de maquietación css y si es necesario usa flexbox y css grid layaout

Preferencias de código:
-No añadas dependencias externas
-Html debe ser semantico
-No uses alertta o ventana modal que aparezca debe tener  el mismo tipo de estilos que la web
-No uses  interhtml, todo contenido debe ser insertado con appchild, o previamente creando el elemento con createelement o previamente
-Cuidado con olvidar prevenir el default en los eventos submit o click
-Prioriza el codigo legible y mantenible.
-Prioriza que el código sea sencillo de entender 
-Si el agente duda, que revise las especificaciones del proyecto y si no que pregunte al usuario.
